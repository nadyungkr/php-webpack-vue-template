## Getting Started

```bash
# PHP Built-in Server
php -S localhost:8080 -t public
# Hot reload (Webpack dev server)
npm run watch
```

## Commands

Name|Description|
----|-----------|
composer run **lint**|*PHPCS* with *[PSR-12](https://www.php-fig.org/psr/psr-12/)*
npm run **lint**|*ESLint*
npm run **build**|Build with *Webpack*
npm run **watch**|*Webpack Dev Server*
